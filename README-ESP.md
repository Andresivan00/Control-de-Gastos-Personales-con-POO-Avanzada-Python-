# Control de Gastos Personales con POO Avanzada (Python)

Un **sistema completo de finanzas personales** hecho con **Programación Orientada a Objetos** moderna, limpio y 100% funcional. Ideal para aprender POO real, manejo de archivos y buenas prácticas.

### Qué hace este código

- Clase `Movimiento`: representa un ingreso o gasto con validaciones estrictas
- Clase `ControlGastos`: el cerebro del sistema con todas las funciones:
  - Registrar ingresos y gastos
  - Calcular saldo actual (ingresos − gastos)
  - Ver resumen de gastos por categoría
  - Guardar y cargar datos en **JSON** y **CSV** (¡persistencia real!)
  - Usar tipado con `typing`, métodos limpios y manejo de errores
